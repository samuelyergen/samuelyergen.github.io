Description de votre projet

Par le groupe le 11/04/2021

Un être maléfique a réveillé une vieille malédiction. Cette dernière a réveillé des monstres qui essaient de vous envahir. Vous n'avez plus le choix, il faut créer une armée afin de défaire les ennemis réveillés par la malédiction d'Ankh.


Attention, ne sous-estimez pas le danger du désert !


Vous commencez le jeu avec de la ressource qui vous permet de faire "spawn" des unités de défenses. Des ennemis se lèvent du sable pour vous envahir, attaquons-les. Vous pouvez ramasser les objets qui "spawn" sur la map, ils vous donneront de la ressource bonus pour vous aider à créer votre armée.


Le code source de l'application est disponible sur le github du projet.
